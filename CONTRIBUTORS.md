If you want to commit any change to this project, your commit must follow these rules:
| Commit Content                      | Emoji Code       |
|-------------------------------------|------------------|
| Bug fix                             | `:wrench:`       |
| New feature                         | `:gift:`         |
| Refactoring                         | `:recycle:`      |
| WIP (Work In Progress)              | `:construction:` |
| Documentation (or text-only commit) | `:memo:`         |
